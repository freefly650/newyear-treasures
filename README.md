# Новорічні Скарби

Next.js застосунок для обліку колекції новорічних іграшок.

## Стек

- Next.js 15 + TypeScript + Tailwind
- Neon/PostgreSQL для даних
- Cloudinary для зображень
- Vercel для деплою

## Поля іграшки

- `Назва`
- `Серія / Набір`
- `Фабрика`
- `Рік` (вільний текст)
- `Покраска` (`Амальгама`, `Фарба`)
- `Рідкість` (`R`, `RR`, `RRR`)
- Нотатки та фото

## Локальний запуск

1. Створіть `.env.local` на основі `.env.example`.
2. Заповніть `DATABASE_URL`, `AUTH_SECRET`, `CLOUDINARY_*`.
3. Встановіть залежності та запустіть:

```bash
npm install
npm run dev
```

## Деплой (Vercel + Neon + Cloudinary)

1. Імпортуйте репозиторій у Vercel.
2. Додайте environment variables з `.env.example`.
3. Вкажіть production `APP_URL`.
4. Задеплойте застосунок.

## Примітки

- eBay функціонал повністю видалено.
- Схема БД створюється/оновлюється автоматично при старті API.
